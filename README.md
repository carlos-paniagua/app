# 食品管理アプリ　

## 環境・技術
- Language：Python
- Version：3.10.11
- Architecture：Flask
- Database:SQLALchemy
- other:HTML5,Bootstrap5

## アプリの機能
- 食品を追加
- 食品の削除機能
- 食品の詳細表示
- 賞味期限通知
- 食品の編集機能
が備わっています．
ユーザー認証とセキュリティ,メール通知,検索機能,カテゴリ分類,レシピやメモ機能,グラフや統計情報,リマインダー機能を追加予定
